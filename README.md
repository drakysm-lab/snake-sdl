**Building**
============
### Requirements
##### *Windows:*
+ mingw32/msys2 with ```gcc build-essential toolchain cmake```.

##### *Linux:*
+ SDL2 ([official repo.](https://github.com/libsdl-org/SDL)); common-build tools.

____________
### Compile:

    $ make build
    $ make run
#### Compile and run (optional): ```$ make test```
